# EcoAndPublicHealthTrendsPlatform

Air quality and climate conditions have been linked to respiratory and cardiovascular disease rates. This project investigates those relationships across B.C. regions and periodic data.

**Project Goals**
- Build a scalable ELT pipeline for multi-source ingestion
- Enable trend analysis across air quality, climate, and disease metrics
- Deliver an interactive dashboard for comparisons

**Data Sources**
- Air Quality — Ministry of Environment
- Climate — Government of Canada Historical Data
- Disease — BC CDC
